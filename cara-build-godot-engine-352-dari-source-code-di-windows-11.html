Cara Build Godot Engine 3.5.2 dari Source Code di Windows 11

RAKIFSUL December 23, 2023 Belajar Build

Apa itu Godot Engine?

Godot Engine atau Godot Game Engine adalah aplikasi game engine.

Apa itu game engine?

Game engine adalah software aplikasi yang digunakan untuk membuat game.

Dalam game engine tersedia sekumpulan tool yang bisa membantu membuat game.

Jadi Godot Engine bisa membantu membuat game.

Apa itu source code?

Source code adalah teks berisi instruksi untuk menjalankan sebuah program, misalnya aplikasi.

Kenapa kita menggunakan Godot Engine?

Godot Engine bisa kita gunakan jika kita hendak membuat game.

Apakah Godot Engine gratis?

Ya. Godot Engine bersifat gratis dan open source.

Karena gratis, kita tidak perlu khawatir untuk membayar aplikasi ini atau melakukan pembagian keuntungan dari game yang kita buat dengan aplikasi ini.

Build, apa itu build?

Build, dalam hal ini, artinya adalah membangun suatu aplikasi siap pakai dari source code nya. Bentuk aplikasi siap pakai ini, di Windows, berformat .exe.

Apakah kita bisa mem-build Godot Engine karena sifatnya open source?

Benar sekali. Karena sifatnya open source, kita bisa mendapatkan source code Godot Engine dan mem-build-nya sendiri.

Apa alasannya kita mem-build Godot Engine sendiri dan bukan menggunakan yang siap pakai saja?

Untuk alasan ini:

  • Mungkin kita ingin berkontribusi dalam pengembangan Godot Engine
  • Mungkin kita ingin membuat modul Godot Engine
  • Mungkin kita ingin menambah fitur Godot Engine
  • Mungkin kita ingin membuat Godot Engine versi kita sendiri

Apa saja persiapan untuk mem-build Godot Engine?

1. Python 3.5+

Install Python 3.5+ di sistem operasi Windows 11 dan pastikan dapat dijalankan melalui terminal di folder manapun.

Oleh karena itu, pastikan Python 3.5+ sudah ditambahkan ke PATH di environment variables.

Cara menambahkan Python 3.5+ ke PATH di environment variables, silakan pelajari sendiri.

Untuk mendapatkan Python 3.5+, kunjungi:

https://www.python.org/

2. Visual Studio Community 2022

Selain Python 3.5+, kita juga perlu Visual Studio Community.

Karena dalam artikel ini yang digunakan adalah versi 2022, maka saya menyarankan untuk menggunakan Visual Studio Community 2022.

Untuk mendapatkannya, silakan kunjungi:

https://visualstudio.microsoft.com/vs/community/

Setelah Visual Studio Community 2022 di-download, jalankan installer-nya, kemudian jangan lupa pilih komponen “Desktop Development with C++” untuk diinstall.

3. SCons 3.0

SCons juga diperlukan untuk mem-build Godot Engine.

Cara meng-install-nya adalah dengan membuka terminal, kemudian jalankan:

python -m pip install scons

4. Source Code Godot Engine

Di artikel ini, saya menggunakan versi LTS dari Godot Engine, yang saat artikel ini ditulis adalah versi 3.5.2.

Jadi, kunjungi:

https://github.com/godotengine/godot/releases

Kemudian dapatkan yang versi 3.5.2-stable:

https://github.com/godotengine/godot/archive/refs/tags/3.5.2-stable.zip

Apakah itu saja persiapannya?

Ya. Sekarang kita siap mem-build Godot Engine.

OK. Bagaimana cara mem-build Godot Engine?

Pertama, ekstrak file zip dari godot-3.5.2-stable.zip yang telah Anda download di GitHub repository-nya.

Selanjutnya, masuk ke folder hasil ekstraknya dengan terminal.

Selanjutnya, jalankan perintah ini:

scons platform=windows

Proses build mungkin akan sedikit lama.

Setelah selesai, kunjungi folder “godot-3.5.2-stable/bin”.

Di sana akan ada file executable dari Godot Engine.

Apakah sudah selesai?

Ya. sekarang Anda tinggal menggunakan executable yang saya sebutkan tadi.

OK. Sekarang saya mengerti. Terima kasih…
