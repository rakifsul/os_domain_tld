Cara Menjalankan Kode PHP di Post dan Page WordPress

RAKIFSUL December 23, 2023 Blog

Apakah WordPress itu?

WordPress adalah sebuah content management system atau yang disingkat sebagai CMS.

WordPress dibuat dengan bahasa pemrograman PHP di sisi servernya.

PHP itu sendiri adalah bahasa pemrograman yang umumnya digunakan di sisi server yang berjalan di atas HTTP server.

Apakah HTTP server itu?

HTTP Server adalah sebuah aplikasi server yang mengolah request menjadi response.

Request datang dari client, kemudian diproses oleh HTTP server, baik dengan maupun tanpa PHP, kemudian hasilnya dikembalikan dalam bentuk response ke client.

Apakah Content Management System itu?

Content management system adalah sebuah software aplikasi yang dibuat untuk memanajemen konten.

Maksudnya adalah aplikasi tersebut dapat melakukan operasi create, read, update, dan delete terhadap konten.

Biasanya CMS digunakan dalam aplikasi berbasis web.

Konsekuensinya, dengan CMS, kita bisa mempermudah operasi CRUD tadi.

CMS yang kami ketahui ada dua jenis:

  • CMS Headful, seperti WordPress, Joomla, Drupal, Moodle, dan lain-lain.
  • CMS Headless, seperti Strapi, Directus, dan lain-lain.

Bandingkan dalam bayangan Anda, jika Anda memanajemen isi website tanpa GUI vs dengan GUI.

WordPress, sebagai CMS, menyediakan GUI untuk memanajemen konten tersebut.

Yang script dan datanya langsung diberikan server setelah request diolah.

Itulah yang dilakukan CMS headful.

CMS Headless itu sendiri adalah sebuah CMS yang pada penggunaannya oleh aplikasi client dilakukan melalui API.

Artinya, walaupun di dalam manajemen kontennya disediakan GUI, saat aplikasi client ingin meminta data, hal tersebut dilakukan melalui API.

Berbeda dengan CMS headful yang mengirimkan script client dan datanya langsung dari server.

Ini akan lebih jelas bagi Anda jika Anda pernah membuat Single Page Application atau SPA.

Apakah istilah Post dalam WordPress itu?

Post adalah sebuah entry dalam database WordPress yang dapat dikategorikan dan dapat diberi tag.

Post bisia di-listing dalam kategori dan tag.

Sebuah post bisa diberikan komentar.

Apakah istilah Page dalam WordPress itu?

Page menyerupai Post, tapi bedanya adalah Page tidak bisa dimasukkan dalam kategori dan tidak bisa diberi tag secara default (tanpa modifikasi WordPress).

Selain itu, Page juga bisa memiliki template.

OK. Sekarang saya mulai paham. Lalu, kenapa kita ingin menjalankan kode PHP di Post dan Page?

Jika Anda ingin menambahkan fitur di dalam website WordPress, maka besar kemungkinan bahwa Anda akan menggunakan plugin.

Makin banyak fitur yang ditambahkan, makin banyak plugin.

Jika semua plugin gratis, mungkin itu tidak masalah. Lalu bagaimana jika tidak?

Dengan kemampuan menjalankan kode PHP di Post dan Page, maka jika user menguasai skill PHP dan WordPress, maka user tidak perlu menginstall banyak plugin untuk banyak fitur.

Cukup dengan coding sendiri.

Untungnya, fitur kode PHP ini juga didapatkan melalui sebuah plugin.

Jadi satu plugin untuk banyak fitur.

Apakah nama plugin yang Anda maksud?

Namanya adalah WPCode Lite.

Untuk menginstallnya, pergi ke menu Plugins > Add New, cari WPCode, kemudian install dan activate.

OK. Sekarang saya sudah menginstallnya. Lalu, bagaimana cara menggunakannya?

Baiklah, saya akan memberi contoh kasus echo “Hello World”.

Dengan menggunakan WPCode Lite, saya akan membuat kode tersebut muncul di Post.

Pertama, buka menu Code Snippets di side menu admin WordPress:

Selanjutnya, klik Add New:

Selanjutnya, pilih ini:

Selanjutnya, pilih code type: “PHP Snippet”:

Isi dengan kode ini:

<?php
echo "Hello World";
?>

Aktifkan dan save snippet:

Pilih shortcode dan save snippet lagi:

Copy shortcode nya.

Kemudian, buat Post baru dan dalam kontennya buat elemen shortcode dengan menggunakan shortcode yang telah di-copy tadi.

OK. Sekarang saya sudah melakukannya. Bagaimana cara mengujinya?

Caranya save Post tadi, buka URL nya dan perhatikan outputnya.

Apakah ada “Hello World”?

Ternyata ada. Apakah saya berhasil?
