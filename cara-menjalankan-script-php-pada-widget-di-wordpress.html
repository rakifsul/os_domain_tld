Cara Menjalankan Script PHP pada Widget di WordPress

RAKIFSUL December 23, 2023 Belajar Web

WordPress adalah sebuah content management system (CMS) yang telah popular sejauh ini.

Dengan menggunakan WordPress, maka pembuatan website menjadi relatif lebih mudah.

Fitur-fitur WordPress sangat banyak, tapi intinya adalah membuat halaman web yang dinamis, tanpa harus melakukan coding.

Apa Itu PHP?

WordPress itu sendiri dibangun dengan menggunakan bahasa pemrograman PHP.

PHP itu sendiri adalah bahasa pemrograman server side yang dapat digunakan untuk membuat aplikasi web maupun website dinamis.

Apa Itu Widget?

Dalam fitur WordPress, ada istilah Widget.

Widget adalah suatu potongan script yang bisa dilekatkan pada layout website berbasis WordPress yang posisinya sudah didaftarkan sebelumnya, tergantung theme apa yang digunakan.

Pada umumnya, widget menggunakan kode HTML, CSS, dan JS.

Benarkah? Apakah widget gambar juga termasuk?

Jawabannya adalah tentu saja.

Gambar yang direpresentasikan dalam sebuah widget pada dasarnya adalah script HTML dengan tag <img>.

Widget PHP

Lalu, apakah memungkinkan bagi kita untuk membuat widget yang menggunakan kode PHP?

Tentu saja, yakni dengan menggunakan sebuah plugin WordPress yang bernama PHP Code Widget.

Bagaimana caranya? Simak terus artikel ini.

Menginstall PHP Code Widget

Karena plugin PHP Code Widget adalah plugin WordPress, maka tambahkan plugin baru terlebih dahulu dengan cara menuju Plugin > Add New.

Kemudian, search “PHP Code Widget”

Kemudian install dan aktifkan.

Saat ini, kita sudah siap untuk menambahkan widget PHP.

Sekarang, silakan menuju Appearance > Widgets.

Tambahkan widget baru bernama PHP Code. Anda bisa search di popup nya dengan mengetikkan “PHP Code”.

Selanjutnya, beri judul (optional) dan isi dengan kode:

<?php
echo "Hello World";
?>

Kemudian klik tombol “Update”.

Sekarang, buka halaman website Anda di mana widget Anda bisa dilihat.

Tampilannya kira-kira seperti gambar di bawah ini, tergantung theme apa yang Anda gunakan.

Akhir Kata
