Cara Download Video YouTube dengan Command Line di Windows 11

RAKIFSUL December 23, 2023 Belajar Windows

YouTube, apakah YouTube itu?

YouTube adalah sebuah platform di mana pengguna bisa menonton atau mempublikasi video.

Platform tersebut saat ini sudah memiliki sangat banyak video dari berbagai topik.

Mulai dari hiburan hingga edukasi.

Download, apakah download itu?

Download adalah sebuah proses transfer data dari server ke komputer client.

Command line, apakah command line itu?

Command line adalah suatu aplikasi yang inputnya berupa teks.

Biasanya digunakan di perangkat desktop atau laptop, walaupun memungkinkan juga untuk perangkat mobile.

Windows 11, apakah Windows 11 itu?

Windows 11 adalah sistem operasi buatan Microsoft yang lebih baru daripada Windows 10.

Sistem operasi ini umumnya digunakan di komputer desktop atau laptop.

OK, saya mulai mengerti. Jadi, apakah yang dimaksud dengan download video YouTube?

Jadi, yang dimaksud dengan download video YouTube adalah proses transfer data video dari server YouTube ke komputer pengguna.

Dengan kata lain, dengan men-download video dari YouTube, kita bisa memperoleh file video yang sama dengan versi online nya.

Hanya saja, file video yang sudah ter-download tidak mengandung iklan dari Adsense.

Sangat menguntungkan bukan?

Apakah hal itu sewajarnya dilakukan?

Menurut saya, ya. Karena, mungkin saja, di lokasi tertentu, kecepatan download cukup rendah sehingga jika ditonton langsung akan terputus-putus.

Alasan lainnya, mungkin karena kita ingin memiliki kopi lokal dari video YouTube tertentu.

Alasan lainnya lagi, mungkin karena kita ingin menonton video YouTube tanpa gangguan iklan.

Jika demikian, solusinya adalah dengan men-download dulu videonya, baru kemudian ditonton.

Kenapa kita mempelajari cara download video YouTube dengan command line? Bukan dengan GUI saja?

Walaupun GUI terkesan lebih cantik dan lebih mudah dipahami, ada kalanya suatu pekerjaan tertentu lebih efisien jika dilakukan dengan command line.

Sebagai contoh kasus adalah saat kita ingin men-download video YouTube .

Dengan command line, proses download video YouTube justru hanya tinggal mengetikkan beberapa karakter dan mem-paste-kan link videonya saja.

Dalam kasus yang lebih advanced, bisa saja proses download dilakukan secara terprogram dengan command line tadi.

OK. Sekarang saya memahami alasannya. Lantas aplikasi command line apakah yang saya perlukan?

Aplikasi yang kita perlukan bernama yt-dlp.

Di manakah saya bisa mendapatkan aplikasi tersebut?

Di https://github.com/yt-dlp/yt-dlp

OK. Tapi… Kenapa di sana hanya ada kumpulan kode?

Untuk men-download exe nya, Anda harus pergi ke halaman release nya:

https://github.com/yt-dlp/yt-dlp/releases

Kemudian download yang ekstensinya exe.

Untuk versi 2023.03.04 , link nya yang ini:

https://github.com/yt-dlp/yt-dlp/releases/download/2023.03.04/yt-dlp.exe

OK. Sudah saya download, selanjutnya, apa yang harus saya lakukan?

Setelah aplikasi itu selesai di-download, masukkan ke folder tertentu.

Sekarang, saya asumsikan bahwa Anda memasukkannya ke folder C:\Install\yt-dlp.exe.

Langkah selanjutnya adalah dengan mendaftarkan folder tersebut ke environment variabel.

Untuk melakukannya klik tanda kaca pembesar di taskbar Windows 11 Anda.

Lalu, cari, dengan cara mengetikkan, “Environment Variable”.

Ketika pilihannya muncul, silakan klik pilihan “Edit the system environment variables”.

Hasilnya seperti ini.

Selanjutnya, klik tombol “Environment Variables” pada window sesuai gambar di atas.

Hasilnya seperti ini.

Selanjutnya, di “System Variables”, pilih “Path” dan double click.

Hasilnya seperti ini.

Klik “New” kemudian “Browse”.

Isi dengan folder C:\Install.

Sampai di sini, proses install sudah selesai. Saatnya menguji.

OK. Bagaimana cara mengujinya?

Cukup mudah, jika Anda mengubah environment variable dalam keadaan command line terbuka, tutup dulu, baru kemudian buka lagi.

Jika tidak, silakan langsung buka command line nya sekarang.

Kemudian ketikkan “yt-dlp” pada command line.

Jika response di command line seperti ini:

Usage: yt-dlp.exe [OPTIONS] URL [URL…]
yt-dlp.exe: error: You must provide at least one URL.
Type yt-dlp --help to see a list of all options.

Maka sudah benar.

Jika ada teks berwarna merah kemungkinan masih salah.

Sebagai catatan, jika environment variable tadi sudah di-setting dengan benar, maka Anda bisa menjalankan yt-dlp dari folder manapun.

OK. Saya sudah berhasil. Bisakah Anda beri saya contoh cara penggunaannya?

Coba ketikkan ini di command line dari folder manapun:

yt-dlp https://www.youtube.com/watch?v=aqz-KE-bpKQ

Silakan, tunggu beberapa saat dan nanti Anda akan melihat ada file video di folder ini.

OK. Ternyata sudah berhasil. Terima kasih!

Sama-sama.
