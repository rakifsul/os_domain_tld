Cara Build Blender 3.6.0 - Aplikasi 3D Editor Open Source

RAKIFSUL December 11, 2023 Blog

Apa itu Windows 11?

Windows 11 adalah versi Windows yang lebih baru daripada Windows 10, Windows 8, Windows 7, Windows Vista, dan Windows XP.

Apa itu Windows?

Windows adalah sistem operasi yang umumnya digunakan untuk pc dan laptop.

Apa itu Blender?

Blender adalah aplikasi 3D editing yang bersifat gratis dan open source.

Apa itu source code?

Source code adalah teks yang berisi instruksi untuk melakukan sesuatu dalam sebuah program dengan bahasa pemrograman tertentu.

Umumnya, source code berupa file teks.

Dengan bahasa pemrograman apa source code Blender 3.6.0 ditulis?

Berdasarkan ekstensi file source code-nya, sebagian besar dengan bahasa pemrograman C dan C++.

Apakah Blender itu gratis?

Ya, seperti yang kami tulis sebelumnya. Blender bersifat gratis dan open source.

Build, apa itu build?

Build, dalam hal ini, artinya adalah membangun suatu aplikasi siap pakai dari source code nya. Bentuk aplikasi siap pakai ini, di Windows, berformat .exe dan mungkin ada juga file .dll-nya.

Kenapa kita perlu mem-build Blender 3.6.0 dari source code?

Untuk alasan ini:

  • Mungkin kita ingin berkontribusi dalam pengembangan Blender 3.6.0
  • Mungkin kita ingin menambah fitur Blender 3.6.0
  • Mungkin kita ingin membuat Blender 3.6.0 versi kita sendiri

Apakah mem-build Blender 3.6.0 itu sulit?

Tergantung.

Jika Anda hanya mengandalkan dokumentasi saja, mungkin sulit.

Tapi, jika Anda bersedia melakukan riset terhadap caranya di search engine atau di forum-forum, mungkin Anda akan mendapatkan petunjuk yang mempermudahnya.

Oleh karena itu, tulisan ini merangkum proses tersebut dalam sebuah artikel.

OK. Jadi, apa saja persiapan untuk build Blender 3.6.0 dari source code di Windows 11?

Pertama, Anda perlu meng-install git.

Walaupun penggunaan git tidak terlihat secara langsung, tetapi mungkin ini akan berguna. Jadi, lebih aman install saja.

Enable “add Git to your PATH” saat di-install.

Untuk men-download git, kunjungi:

https://git-scm.com/downloads

Kedua, Anda perlu meng-install svn client.

Gunakan SilkSVN, karena ini yang kami pakai.

SVN Client digunakan untuk men-download dependencies dari source code Blender.

Untuk men-download SilkSVN, kunjungi:

https://sliksvn.com/download/

Ketiga, Anda perlu meng-install CMake.

CMake berguna untuk membuat file platform-specific project file yang cross platform.

Karena kita menggunakan Windows, maka CMake akan membuatkan file project yang compatible dengan Visual Studio Community 2022.

Sekali lagi, mungkin penggunaannya tidak terlihat secara langsung, tapi ini benar-benar dibutuhkan.

Enable “Add CMake to the system PATH for all users” saat di-install.

Untuk mendapatkannya, kunjungi:

https://cmake.org/download/

Keempat, Anda perlu meng-install Visual Studio Community 2022

Walaupun ukurannya cukup besar, kita benar-benar memerlukan ini, karena prosess build akan dilakukan dengan ini.

Enable “Desktop Development with C++” Saat di-install.

Untuk mendapatkannya, kunjungi:

https://visualstudio.microsoft.com/free-developer-offers/

Kelima, Anda harus mendapatkan source code Blender versi 3.6.0.

Untuk mendapatkannya, kunjungi:

https://download.blender.org/source/blender-3.6.0.tar.xz

Keenam, Anda harus meng-install 7-Zip.

7-Zip digunakan untuk mengekstrak file .tar.xz dari source code Blender tadi.

Untuk mendapatkannya, kunjungi:

https://www.7-zip.org/

OK. Saya sudah meng-install itu semua. Selanjutnya apa?

Langkah pertama, buat sebuah folder bernama “Blender-3.6.0-Mod”.

Sebenarnya, namanya bebas. Ini hanya untuk kesesuaian dalam tutorial ini agar lebih mudah.

Masukkan file source code Blender ke dalam folder ini, kemudian ekstrak dengan 7-Zip.

Ekstrak pertama akan menghasilkan .tar, ekstrak lagi hingga hasilnya folder.

folder ini bernama “blender-3.6.0”.

Langkah kedua, buat folder bernama “lib” di dalam folder “Blender-3.6.0-Mod”.

Jadi, posisi folder “lib” ini sejajar dengan folder “blender-3.6.0”.

Selanjutnya, masuk ke dalam folder “lib” dan jalankan perintah ini dengan PowerShell atau CMD:

svn checkout https://svn.blender.org/svnroot/bf-blender/tags/blender-3.6-release/lib/win64_vc15/

Ini akan men-download dependency Blender 3.6.0 dan ukurannya cukup besar. Jadi akan agak lama.

Langkah ketiga, masuklah ke dalam folder “blender-3.6.0” dengan CMD.

WAJIB dengan CMD, tidak bisa dengan PowerShell, karena kita akan menjalankan perintah batch di sini.

Langkah keempat, jalankan perintah ini dengan CMD tadi:

make

Proses build akan dimulai. Ini akan lama. Silakan tunggu.

Setelah selesai, Anda bisa mendapatkan executables dan binary dari Blender di folder “build_windows_x64_vc17_Release/bin/Release”.

Langkah kelima, tidak wajib, Anda bisa membuat file installer untuk Blender.

Caranya, Anda bisa menggunakan NSIS, tapi kami tidak membahasnya di sini.

Untuk mendapatkan NSIS, kunjungi:

https://nsis.sourceforge.io/Download

Silakan pelajari sendiri.

OK. Sekarang saya paham. Terima kasih ya…

Sama-sama.
