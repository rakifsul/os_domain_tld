Mengenal yt-dlp dan Cara Meng-install-nya di Ubuntu

RAKIFSUL December 23, 2023 Belajar Linux

Apa itu Ubuntu?

Ubuntu adalah sistem operasi yang bersifat gratis dan open source.

Kenapa kita menggunakan Ubuntu?

Ubuntu adalah sistem operasi yang gratis, maka untuk mendapatkannya kita tidak perlu membayar.

Sehingga, kita bisa menikmati versi original dari sistem operasi tersebut secara cuma-cuma.

Selain itu, Ubuntu berbasis Linux yang keamanannya lebih ketat, sehingga jarang terkena malware.

Apa itu yt-dlp?

yt-dlp adalah sebuah aplikasi command line, yang digunakan melalui terminal, untuk men-download video dari YouTube.

Aplikasi itu bersifat gratis dan open source juga.

Sehingga, kita tidak perlu membayar untuk mendapatkannya.

Apa fitur yt-dlp yang paling berguna?

Fitur yt-dlp yang paling berguna adalah download video secara single, seluruh isi playlist, bahkan seluruh isi channel.

Selain itu, masih banyak fitur lain yang tidak dijelaskan dalam artikel ini.

Kenapa kita belajar cara meng-install-nya?

Walaupun sebenarnya aplikasi yt-dlp bersifat standalone, artinya, cukup download binary-nya, tapi mungkin diperlukan trik khusus agar kita bisa dapat mengaksesnya dari folder manapun di terminal Ubuntu.

Kenapa kita belajar cara meng-install-nya di Ubuntu?

Alasan pertama adalah: Ubuntu merupakan sistem operasi yang bersifat gratis dan open source.

Alasan kedua adalah: yt-dlp merupakan aplikasi download YouTube yang bersifat gratis dan open source.

Artinya, dari kedua alasan tadi, kita bisa men-download video YouTube secara gratis tanpa harus membayar.

Selain itu, alasan lainnya adalah, kita perlu perintah khusus di terminal Ubuntu yang mungkin agak asing bagi orang yang tidak terbiasa menggunakan Ubuntu atau Linux.

OK. Sekarang saya paham. Lalu, bagaimana cara meng-install yt-dlp di Ubuntu?

Begini.

Pertama-tama sadarilah terlebih dahulu, bahwa yt-dlp di-host di repository GitHub ini:

https://github.com/yt-dlp/yt-dlp

Untuk versi binary-nya, ada di sini:

https://github.com/yt-dlp/yt-dlp/releases

Dan versi binary terbarunya, diakses via URL ini:

https://github.com/yt-dlp/yt-dlp/releases/latest

Maka, untuk men-downloadnya via terminal (langkah 1), jalankan perintah ini:

sudo wget -qO /usr/local/bin/yt-dlp https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp

Arti dari perintah itu adalah kita men-download binary yt-dlp versi terbaru, kemudian menempatkannya di folder “/usr/local/bin”.

Karena ada di folder tadi, maka kita tidak perlu mendaftarkan environment variable atau PATH tambahan untuk bisa mengaksesnya dari folder manapun saat yt-dlp ingin digunakan.

langkah selanjutnya (langkah 2), kita memberi izin execute untuk binary tadi:

sudo chmod a+rx /usr/local/bin/yt-dlp

Sekarang, kita bisa mencobanya.

Untuk memastikan bahwa yt-dlp telah ter-install dengan benar, jalankan perintah ini:

yt-dlp --version

Jika benar, maka versi yt-dlp akan muncul.

Untuk mencoba download video, jalankan perintah ini:

yt-dlp [url-video]

Misal:

yt-dlp https://www.youtube.com/watch?v=aqz-KE-bpKQ&pp=ygUOYmlnIGJ1Y2sgYnVubnk%3D

Perintah di atas adalah untuk men-download video “Big Buck Bunny” ke folder di mana yt-dlp dijalankan.

Tunggu beberapa saat.

Jika sudah selesai, Anda boleh mencoba memainkan video tadi dengan video player Anda.

OK. Sekarang saya mengerti. Terima kasih…

Sama-sama.
